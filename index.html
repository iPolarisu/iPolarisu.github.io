Discord Bot

U-Cursed

Discord bot that scraps the U-Cursos website (FCFM - Universidad de Chile) to get information about available Engineering courses via different commands.

At the moment the Bot is being hosted at Heroku, not optimal but gets the job done for now (if dynos are available). To invite it you need to use this invitation.

For more information and self hosting, you can check the project's GitHub repo.

Website

STIMA.ME

Website that calculates how much value you are getting for every dollar invested in a Steam game (2019 static data). The name of the website is a pun in spanish (estimate me).

Videogame

Retry?

Retry? is an arcade/roguelike/bullethell/space-shooter indie videogame with a heavy emphasis on resource management. It has a cool twist in its core mechanic and a meta narrative.

It was started as a project for a videogame workshop at university but its development may be continued even though it will be slower.

For more information you can check the itch.io page.
